라쿤 로봇 언플러그드 활동 20가지

컴퓨터 없이 로봇 팔을 손으로 직접 움직여 동작을 가르쳐요! '언플러그드 모드'는 위치와 그리퍼(집게) 상태를 기록하고, 재생 버튼만 누르면 로봇이 그대로 따라하는 라쿤 로봇의 핵심 기능입니다. 아래 활동들을 통해 로봇 제어의 기본 원리를 재미있게 배워보세요.

📦 활동 1: 원기둥 하나를 A에서 B로 옮기기

▼

활동 내용: 로봇 팔을 손으로 움직여 원기둥을 잡고 (그리퍼 닫힘) A 지점에서 B 지점으로 옮긴 후 놓는 (그리퍼 열림) 과정을 기록하고 재생합니다.

[여기에 로봇 팔이 원기둥을 A에서 B로 옮기는 이미지를 넣으면 학습 효과가 좋습니다]

💡핵심 기능

  • ✅ 위치 기록: 로봇 팔의 특정 위치를 기억시키는 기본 기능
  • ✅ 그리퍼 제어: 물체를 잡고(닫힘) 놓는(열림) 동작 기록

🧱 활동 2: 두 개의 원기둥을 쌓기

▼

활동 내용: 첫 번째 원기둥을 A에서 B로 옮긴 후, 다시 두 번째 원기둥을 가져와 첫 번째 원기둥 위에 쌓는 복합적인 운반 및 정렬 활동입니다. 각 단계의 팔 위치와 그리퍼 상태를 기록합니다.

[여기에 로봇 팔이 원기둥 두 개를 차곡차곡 쌓는 이미지를 넣으면 학습 효과가 좋습니다]

💡핵심 기능

  • ✅ 연속된 위치 기록: 여러 단계를 순서대로 기록하는 능력
  • ✅ 그리퍼 제어:

🔢 활동 3: 여러 물체를 순서대로 다른 위치로 옮겨 놓기

▼

활동 내용: 세 개 이상의 작은 물체들을 특정 순서에 따라 지정된 다른 위치로 이동시키는 활동입니다. 각 물체를 잡고 (그리퍼 닫힘) 원하는 위치에 놓는 (그리퍼 열림) 과정을 기록합니다.

💡핵심 기능

  • ✅ 다단계 운반: 복잡한 이동 경로 계획 및 실행
  • ✅ 순서 지정: 작업의 순서(알고리즘) 개념 학습
  • ✅ 그리퍼 제어:

🎨 활동 4: 크기나 색깔이 다른 물체들을 분류하기

▼

활동 내용: 로봇 팔을 이용해 다양한 물체를 집어 올리고 (그리퍼 닫힘) 분류 기준에 따라 정해진 위치에 놓는 (그리퍼 열림) 활동입니다.

💡핵심 기능

  • ✅ 물체 분류: 조건에 따른 분류 개념 학습
  • ✅ 여러 위치 기록: 각 분류 장소의 위치를 모두 기록

✍️ 활동 5: 로봇 팔로 특정 도형 그리기

▼

활동 내용: 그리퍼를 사용하지 않고 로봇 팔의 관절들을 손으로 직접 움직여 공중에서 'ㅁ' 또는 'ㅅ'과 같은 간단한 도형 모양의 궤적을 기록하고 재생하는 활동입니다.

[여기에 로봇 팔이 허공에 사각형 궤적을 그리는 이미지를 넣으면 학습 효과가 좋습니다]

🔑핵심 기능

  • ✅ 궤적 기록: 점(위치)들이 모여 선(궤적)이 되는 원리 이해
  • ✅ 팔 관절 제어: 그리퍼 없이 팔의 움직임에만 집중

🔄 활동 6: 물건을 집어 올린 후 제자리로 내려놓는 반복 활동

▼

활동 내용: 그리퍼로 물건을 잡았다가(그리퍼 닫힘) 일정 높이까지 들어 올린 후 다시 원래 위치에 내려놓는(그리퍼 열림) 과정을 여러 번 반복하여 기록하는 활동입니다.

💡핵심 기능

  • ✅ 반복 동작 기록: 자동화의 기본인 반복 작업 학습
  • ✅ 그리퍼 제어

🚧 활동 7: 장애물을 피해 물체를 운반하기

▼

활동 내용: 테이블 위에 놓인 작은 장애물(예: 블록)을 넘어서 물체를 A 지점에서 B 지점으로 옮기는 활동입니다. 로봇 팔의 관절 움직임(특히 상하 움직임)을 섬세하게 조절하여 기록합니다.

[여기에 로봇 팔이 블록 장애물을 아치형으로 넘어 물건을 옮기는 이미지를 넣으면 학습 효과가 좋습니다]

🔑핵심 기능

  • ✅ 정밀한 관절 제어: 3차원 공간에서의 경로 설정
  • ✅ 장애물 회피 움직임 기록: 문제 해결 능력 향상

👋 활동 8: 로봇 팔로 '인사'하는 동작 만들기

▼

활동 내용: 로봇 팔을 손으로 움직여 위아래로 흔들거나 그리퍼를 열고 닫으며 인사하는 듯한 재미있는 동작을 기록하고 재생합니다.

💡핵심 기능

  • ✅ 창의적 움직임 기록: 상상력을 발휘하여 동작 만들기
  • ✅ 그리퍼 및 관절 활용: 다양한 부품을 활용한 표현

📥 활동 9: 작은 물체들을 정해진 용기에 하나씩 담기

▼

활동 내용: 여러 개의 작은 구슬이나 블록을 하나씩 집어 들어 컵이나 상자와 같이 입구가 좁은 용기 안에 정확히 떨어뜨리는 활동입니다.

🔑핵심 기능

  • ✅ 정교한 집기 및 놓기: 소근육 발달과 유사한 정밀 제어
  • ✅ 정확한 위치 기록: 목표 지점의 정확성 향상

↩️ 활동 10: 물체를 옮긴 후 초기 위치로 복귀하기

▼

활동 내용: 물체 하나를 A에서 B로 옮긴 후, 로봇 팔을 다시 처음 시작했던 초기 위치로 되돌리는 과정을 기록하는 활동입니다.

💡핵심 기능

  • ✅ 초기 위치 복귀(Homing): 자동화 공정의 기본 사이클 이해
  • ✅ 연속 동작 기록

⭕ 활동 11: 물체를 원형 경로를 따라 이동시키기

▼

활동 내용: 로봇 팔이 물체를 잡고 시작점에서 원형 또는 곡선 형태의 경로를 따라 지정된 종료 지점까지 이동하는 과정을 기록하고 재생합니다.

🔑핵심 기능

  • ✅ 곡선 궤적 기록: 여러 개의 점을 연결하여 곡선 구현
  • ✅ 그리퍼 제어

↕️ 활동 12: 그리퍼를 사용하여 특정 높이에서 물체를 잡고 놓는 연습

▼

활동 내용: 물체를 지면에서 특정 높이(예: 5cm, 10cm)까지 들어 올린 후 내려놓는, 또는 다른 높이의 지지대에서 물체를 잡고 놓는 반복 연습 활동입니다.

💡핵심 기능

  • ✅ 그리퍼의 정교한 사용
  • ✅ 수직 이동 기록: Z축 좌표에 대한 개념 학습

🔡 활동 13: 로봇 팔로 공중에 글자나 숫자 쓰기

▼

활동 내용: 로봇 팔을 손으로 움직여 공중에 'A', '1'과 같은 간단한 글자나 숫자의 획을 따라가며 위치를 기록하고 재생합니다. (그리퍼는 닫지 않음).

🔑핵심 기능

  • ✅ 정밀한 궤적 기록
  • ✅ 팔 관절 제어

↔️ 활동 14: 두 지점 사이를 왕복하며 물체 운반하기

▼

활동 내용: 하나의 물체를 A에서 B로 옮긴 후, 다시 B에서 A로 옮기는 과정을 반복하여 기록하는 활동입니다.

💡핵심 기능

  • ✅ 왕복 운반: 물류 시스템의 기본 동작 이해
  • ✅ 반복 동작 기록

🌊 활동 15: 여러 관절을 동시에 사용하여 유연한 곡선 움직임 만들기

▼

활동 내용: 4개의 관절 중 여러 개를 동시에 손으로 움직여 정해진 지점까지 가장 유연하고 부드러운 곡선 형태의 궤적을 만들어 기록하고 재생하는 활동입니다.

🔑핵심 기능

  • ✅ 다중 관절 제어(보간): 로봇 움직임의 자연스러움 이해
  • ✅ 부드러운 궤적 기록

🚂 활동 16: 작은 블록으로 '기차' 모양 만들기

▼

활동 내용: 여러 개의 작은 블록을 하나씩 옮겨 일렬로 길게 늘어놓거나 서로 연결하여 '기차' 형태를 만드는 활동입니다.

💡핵심 기능

  • ✅ 연속된 물체 배열: 조립 및 생산 라인의 기본 원리 체험
  • ✅ 그리퍼 제어

👉 활동 17: 그리퍼를 이용해 물건을 살짝 '밀어내는' 동작

▼

활동 내용: 그리퍼를 열지 않고 닫은 상태에서 로봇 팔을 움직여 물건을 살짝 건드려 밀어내는 듯한 동작을 기록하고 재생합니다.

🔑핵심 기능

  • ✅ 그리퍼 닫힘 상태에서의 팔 움직임: 도구의 다른 사용법 탐구

👆 활동 18: 로봇 팔로 제스처 취하기 (예: 손가락으로 가리키기)

▼

활동 내용: 로봇 팔과 그리퍼를 특정 방향으로 움직여 마치 사람 손가락처럼 무언가를 가리키는 제스처를 만들고 기록합니다.

💡핵심 기능

  • ✅ 의인화된 동작 기록: 로봇과의 상호작용 및 감성적 접근
  • ✅ 그리퍼 및 관절 활용

👐 활동 19: 정해진 간격으로 그리퍼 열고 닫기

▼

활동 내용: 언플러그드 모드는 시간 기록 기능은 없으므로, 사용자가 '마음속으로' 시간을 세며 일정 간격으로 그리퍼를 열고 닫는 반복 동작을 여러 번 기록하는 활동입니다.

🔑핵심 기능

  • ✅ 그리퍼 상태 반복 기록: 시간 개념을 제외한 순수 동작 반복 연습

📏 활동 20: 테이블 위 점선을 따라 물체 옮기기

▼

활동 내용: 테이블에 그려진 점선 경로를 따라 물체를 정확히 이동시키는 활동입니다. 로봇 팔의 움직임과 그리퍼 사용을 정밀하게 기록하여 선을 벗어나지 않도록 합니다.

[여기에 점선 위를 따라가는 로봇 팔의 그리퍼 끝부분을 클로즈업한 이미지를 넣으면 좋습니다]

💡핵심 기능

  • ✅ 정밀 경로 추적: 산업용 로봇의 '티칭'과 가장 유사한 활동
  • ✅ 그리퍼 제어
