📖언플러그드 모드 사용 방법

  1. 1.
    전원 켜기: 전원 버튼을 한 번 눌러 로봇의 전원을 켭니다.
  2. 2.
    모드 진입: 티칭 버튼을 한 번 누르면 언플러그드 모드로 진입합니다.
  3. 3.
    위치 설정: 로봇 팔을 손으로 직접 움직여 원하는 위치로 이동시킵니다.

    4. ⚠️주의! 너무 세게 움직이면 고장날 수 있어요.

  4. 4.
    위치 기록: 티칭 버튼을 한 번 누르면 현재 위치가 기록됩니다.
  5. 5.
    그리퍼 기록: 티칭 버튼을 두 번 누르면 그리퍼 상태가 바뀝니다. (열림 ↔ 닫힘)
  6. 6.
    동작 재생: 재생 버튼을 누르면 기록된 동작을 순서대로 재생합니다.
  7. 7.
    기록 삭제: 삭제 버튼을 한 번 누르면 마지막 동작이, 길게 누르면 모든 동작이 삭제됩니다.
