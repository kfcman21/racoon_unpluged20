라쿤 로봇 언플러그드 완전 정복

컴퓨터 없이 로봇 팔을 손으로 직접 움직여 동작을 가르쳐요! '언플러그드 모드'는 위치와 그리퍼(집게) 상태를 기록하고, 버튼 하나로 로봇이 그대로 따라하게 만드는 라쿤 로봇의 핵심 기능입니다.

📖언플러그드 모드 사용 방법

  1. 1.
    전원 켜기: 전원 버튼을 한 번 눌러 로봇의 전원을 켭니다.
  2. 2.
    모드 진입: 전원이 켜진 상태에서 티칭 버튼을 한 번 누르면 언플러그드 모드로 진입합니다.
  3. 3.
    위치 설정: 로봇 팔을 손으로 직접 움직여 원하는 위치로 이동시킵니다. 로봇 몸체를 한 손으로 잡고 움직이면 안정적입니다.

    4. ⚠️주의하세요!

    로봇 팔을 너무 세게 움직이면 고장의 원인이 될 수 있습니다. 부드럽게 다뤄주세요.

  4. 4.
    위치 기록: 원하는 위치에서 티칭 버튼을 한 번 누르면 현재 위치가 기록됩니다.
  5. 5.
    그리퍼 기록: 그리퍼 상태를 바꾸며 기록하려면 티칭 버튼을 두 번 누릅니다. (열림 ↔ 닫힘)
  6. 6.
    동작 반복: 3~5번 과정을 반복하여 여러 동작을 순서대로 추가 기록할 수 있습니다.
  7. 7.
    동작 재생: 재생 버튼을 누르면 기록된 동작을 처음부터 순서대로 재생합니다.
  8. 8.
    기록 삭제: 삭제 버튼을 한 번 누르면 마지막 동작이, 길게 누르면 모든 동작이 삭제됩니다.
  9. 9.
    초기 위치 복귀: 전원 버튼을 한 번 누르면 로봇이 초기 위치로 돌아갑니다.

[여기에 로봇 팔이 원기둥을 A에서 B로 옮기는 이미지를 넣으면 학습 효과가 좋습니다]

💡핵심 기능

  • ✅ 위치 기록: 티칭 버튼 한 번 클릭
  • ✅ 그리퍼 제어: 티칭 버튼 두 번 클릭

🧱 활동 2: 두 개의 원기둥을 쌓기

▼

활동 내용: 첫 번째 원기둥을 A에서 B로 옮긴 후, 다시 원기둥 하나를 A에 놓고 두 번째 원기둥을 가져와 첫 번째 원기둥 위에 쌓는 복합적인 운반 및 정렬 활동입니다.

[여기에 로봇 팔이 원기둥 두 개를 차곡차곡 쌓는 이미지를 넣으면 학습 효과가 좋습니다]

💡핵심 기능

  • ✅ 연속된 위치 기록: 여러 단계를 순서대로 기록하는 능력
  • ✅ 그리퍼 제어

🧩 활동 17: 간단한 조립 동작 시뮬레이션

▼

활동 내용: 두 개의 블록 중 하나를 잡고(그리퍼 닫힘) 다른 블록 위에 정확하게 끼워 넣는 듯한 조립 동작을 시뮬레이션하여 기록합니다.

[여기에 로봇 팔이 레고 블록 두 개를 조립하는 이미지를 넣으면 학습 효과가 좋습니다]

🔑핵심 기능
